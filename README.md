# Viral Intel
Daily viral content intelligence dashboard. Monitors X/Twitter, TikTok, Instagram, and YouTube for trending content, analyzes engagement patterns, and generates actionable content frameworks.

## Features
- Topic management with keyword tracking
- Daily automated scans via scheduled task
- Pattern analysis (hooks, emotional triggers, formats)
- Content template generation
- Telegram delivery integration

## Stack
- React + TypeScript + Vite
- Express backend
- Tailwind CSS + shadcn/ui
- Drizzle ORM

## Setup
```
npm install
npm run dev
```
